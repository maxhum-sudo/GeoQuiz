# GeoQuiz

A simple geography game where you drag four cities into the correct ranking based on a surprise metric.

## How It Works

Each round asks you to rank four cities by a hidden metric such as:

- Population
- Restaurants
- Cafes
- Median household income
- UHNW individuals
- Median home price
- Median monthly rent

Values stay hidden until you submit your answer.

## Tech Stack

- Next.js
- React
- TypeScript
- `@dnd-kit` for drag and drop

## Getting Started

```bash
npm install
npm run dev
```

Then open [http://127.0.0.1:3000](http://127.0.0.1:3000).
